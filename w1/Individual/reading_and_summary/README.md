## 對於 LangChain 的理解

1. 本身非完整的後端 server，而是一個框架。
2. 具有將大型語言模型與其他資源整合的能力，例如：
   - 資料庫
   - 外部 API
   - 其他平台的應用 (如 Google Calendar)
3. 支持任務拆解與流程自動化，例如建立一個多頁面的網站時，可以自動逐步完成每一個頁面。
4. 具備長期記憶功能，能夠記住使用者的喜好與背景資料。
5. 自動完成後續的工作，例如 IFTTT (If This Then That)。
